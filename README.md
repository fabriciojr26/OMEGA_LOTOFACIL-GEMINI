Lotofácil Ômega v2.0 — Sistema de Geração EstratégicaEste repositório contém o código-fonte completo do Lotofácil Ômega v2.0, um sistema projetado para gerar portfólios de jogos para a Lotofácil com base em análises probabilísticas e heurísticas avançadas.ArquiteturaO sistema é dividido em duas partes principais, otimizadas para deploy em plataformas gratuitas e de alta performance:frontend/: Uma Single-Page Application (SPA) construída com HTML, CSS e JavaScript puros. É responsável pela interface do usuário, interação, e pela nova funcionalidade de Conferidor de Resultados. Projetado para ser publicado no Netlify.backend/: Uma API robusta desenvolvida com FastAPI (Python). Contém o núcleo algorítmico (SIPSA-lite), a lógica de geração de portfólios, os guard-rails de diversidade (Hamming, Exposição de Pares) e o novo endpoint de R adaptativo. Projetado para ser publicado no Coyeb via Docker.Funcionalidades PrincipaisGeração Ancorada: Cria jogos baseados na repetição de dezenas do último sorteio.R Adaptativo: Sugere a faixa de repetição ideal com base na análise histórica recente.Heurísticas MEPA/CAIC: Filtra combinações para garantir que os jogos sigam padrões estruturais comuns.Portfólio Otimizado: Seleciona os melhores jogos e aplica regras de diversidade para maximizar a cobertura.Conferidor de Resultados: Permite ao usuário verificar o desempenho de um portfólio gerado contra um resultado real.Como Publicar (Deploy)Siga o guia de publicação detalhado no arquivo GUIA_DEPLOY.md.Este software é uma ferramenta de estudo e análise e não garante ganhos financeiros. Jogue com responsabilidade.
